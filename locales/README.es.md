## 📜 Descripción
**TrollMind** es un juego de rompecabezas moderno y atrevido diseñado para desafiar tu ingenio contra un troll travieso.
Construida de forma nativa para Android utilizando **Jetpack Compose** y **Material 3**, la aplicación ofrece una interfaz de usuario vibrante e interactiva con animaciones fluidas.
El proyecto sigue la arquitectura **MVVM (Model-View-ViewModel)** y utiliza **Koin 4.0** para una inyección de dependencias robusta, asegurando que el código sea limpio, testable y escalable.

## ✨ Características
- 😈 **Burla al Troll:** Mecánicas de rompecabezas interactivas con reacciones dinámicas del troll.
- 🎨 **Interfaz Material 3:** Diseño moderno y receptivo basado en maquetas de Figma de alta fidelidad.
- ⚡ **Animaciones Avanzadas:** Mascota SVG personalizada con animaciones de flotación y pulso.
- 🌓 **Temas Dinámicos:** Cambio fluido entre los modos claro y oscuro.
- 🌐 **Alcance Global:** Soporte completo para **25 idiomas** (incluyendo persa, inglés, chino, ruso, árabe, etc.) con soporte automático para diseño RTL.
- 📊 **Seguimiento del Progreso:** Sistema de progresión por niveles con un tablero visual.
- 🏗️ **Arquitectura Limpia:** Construido con MVVM, StateFlow y Koin 4.0 para un estándar de desarrollo premium.
- 💾 **Persistencia Robusta:** Aprovechando **Room Database** y **DataStore** para un almacenamiento de datos confiable.
