## 📜 Description
**TrollMind** est un jeu de puzzle moderne et espiègle conçu pour défier votre esprit contre un troll malicieux.
Construite nativement pour Android avec **Jetpack Compose** et **Material 3**, l'application offre une interface utilisateur vibrante et interactive avec des animations fluides.
Le projet suit l'architecture **MVVM (Model-View-ViewModel)** et utilise **Koin 4.0** pour une injection de dépendances robuste, garantissant un code propre, testable et évolutif.

## ✨ Caractéristiques
- 😈 **Déjouez le Troll :** Mécaniques de puzzle interactives avec des réactions dynamiques du troll.
- 🎨 **Interface Material 3 :** Design moderne et réactif basé sur des maquettes Figma de haute fidélité.
- ⚡ **Animations Avancées :** Mascotte SVG personnalisée avec des animations de flottaison et de pulsation.
- 🌓 **Thèmes Dynamiques :** Passage fluide entre les modes clair et sombre.
- 🌐 **Portée Mondiale :** Support complet pour **25 langues** (y compris le persan, l'anglais, le chinois, le russe, l'arabe, etc.) avec support automatique de la mise en page RTL.
- 📊 **Suivi de la Progression :** Système de progression par niveaux avec un tableau de bord visuel.
- 🏗️ **Architecture Propre :** Construit avec MVVM, StateFlow et Koin 4.0 pour un standard de développement premium.
- 💾 **Persistance Robuste :** Utilisation de **Room Database** et **DataStore** pour un stockage de données fiable.

## 🛠 Construit avec

| Catégorie | Technologie |
| :--- | :--- |
| 🏛 Architecture | MVVM (Model-View-ViewModel) |
| 🖼️ Framework UI | [Jetpack Compose](https://developer.android.com/jetpack/compose) (Material 3) |
| 🛠️ Injection de dépendances | [Koin 4.0](https://insert-koin.io/) |

| 📦 Stockage local | [Room Database](https://developer.android.com/training/data-storage/room) & [DataStore](https://developer.android.com/topic/libraries/architecture/datastore) |
