## 📜 विवरण
**TrollMind** (जिसे **TrapTap** के रूप में भी जाना जाता है) एक आधुनिक पहेली गेम है जिसे एक शरारती ट्रोल के खिलाफ आपकी बुद्धि को चुनौती देने के लिए डिज़ाइन किया गया है।
**Jetpack Compose** और **Material 3** का उपयोग करके एंड्रॉइड के लिए मूल रूप से बनाया गया, ऐप सहज एनिमेशन के साथ एक जीवंत और इंटरैक्टिव यूआई प्रदान करता है।
यह प्रोजेक्ट **MVVM (Model-View-ViewModel)** आर्किटेक्चर का पालन करता है और मजबूत डिपेंडेंसी इंजेक्शन के लिए **Koin 4.0** का उपयोग करता है।

## ✨ विशेषताएं
- 😈 **ट्रोल को मात दें:** गतिशील ट्रोल प्रतिक्रियाओं के साथ इंटरैक्टिव पहेली मैकेनिक्स।
- 🎨 **Material 3 UI:** उच्च-निष्ठा फिग्मा मॉक के आधार पर आधुनिक, उत्तरदायी डिजाइन।
- ⚡ **उन्नत एनिमेशन:** फ्लोट और पल्स एनिमेशन के साथ कस्टम-रेंडर किया गया SVG शुभंकर।
- 🌓 **डायनामिक थीमिंग:** लाइट और डार्क मोड के बीच निर्बाध स्विचिंग।
- 🌐 **वैश्विक पहुंच:** स्वचालित RTL लेआउट समर्थन के साथ **25 भाषाओं** (फारसी, अंग्रेजी, चीनी, रूसी, अरबी, आदि सहित) के लिए पूर्ण समर्थन।
- 📊 **प्रगति ट्रैकिंग:** विजुअल डैशबोर्ड के साथ स्तर-आधारित प्रगति प्रणाली।
- 🏗️ **क्लीन आर्किटेक्चर:** प्रीमियम विकास मानक के लिए MVVM, StateFlow और Koin 4.0 के साथ निर्मित।
- 💾 **मजबूत डेटा स्टोरेज:** विश्वसनीय डेटा भंडारण के लिए **Room Database** और **DataStore** का लाभ उठाना।

## 🛠 इनके साथ निर्मित

| श्रेणी | तकनीक |
| :--- | :--- |
| 🏛 आर्किटेक्चर | MVVM (Model-View-ViewModel) |
| 🖼️ UI फ्रेमवर्क | [Jetpack Compose](https://developer.android.com/jetpack/compose) (Material 3) |
| 🛠️ डिपेंडेंसी इंजेक्शन | [Koin 4.0](https://insert-koin.io/) |
| 🔄 स्टेट मैनेजमेंट | StateFlow & SharedFlow |
| 🧭 नेविगेशन | [Compose Navigation](https://developer.android.com/jetpack/compose/navigation) |
| 🔄 कॉरौटाइन्स | [Kotlinx Coroutines](https://github.com/Kotlin/kotlinx.coroutines) |
| 📦 सीरियलाइजेशन | [Kotlinx Serialization](https://github.com/Kotlin/kotlinx.serialization) |
| 📦 स्थानीय स्टोरेज | [Room Database](https://developer.android.com/training/data-storage/room) & [DataStore](https://developer.android.com/topic/libraries/architecture/datastore) |
