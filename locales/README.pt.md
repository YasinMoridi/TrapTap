## 📜 Descrição
**TrollMind** (também conhecido como **TrapTap**) é um jogo de quebra-cabeça moderno e atrevido, projetado para desafiar sua inteligência contra um troll travesso.
Construído nativamente para Android usando **Jetpack Compose** e **Material 3**, o aplicativo oferece uma interface vibrante e interactive com animações suaves.
O projeto segue a arquitetura **MVVM (Model-View-ViewModel)** e utiliza **Koin 4.0** para uma injeção de dependências robusta.

## ✨ Funcionalidades
- 😈 **Seja mais esperto que o Troll:** Mecânicas de quebra-cabeça interativas com reações dinâmicas.
- 🎨 **Interface Material 3:** Design moderno e responsivo baseado em modelos de alta fidelidade do Figma.
- ⚡ **Animações Avanzadas:** Mascote SVG personalizado com animações de flutuação e pulso.
- 🌓 **Tematização Dinâmica:** Troca contínua entre os modos Claro e Escuro.
- 🌐 **Alcance Global:** Suporte total para **25 idiomas** (incluindo Persa, Inglês, Chinês, Russo, Árabe, etc.) com suporte automático a layout RTL.
- 📊 **Acompanhamento de Progresso:** Sistema de progressão baseado em níveis com um painel visual.
- 🏗️ **Arquitetura Limpa:** Construído com MVVM, StateFlow e Koin 4.0 para um padrão de desenvolvimento premium.
- 💾 **Persistência Robusta:** Utilizando **Room Database** e **DataStore** para armazenamento de dados confiável.
