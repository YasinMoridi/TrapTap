## 📜 Descriere
**TrollMind** (cunoscut și sub numele de **TrapTap**) este un joc de puzzle modern și îndrăzneț, conceput pentru a-ți pune la încercare inteligența împotriva unui troll poznaș.
Construit nativ pentru Android folosind **Jetpack Compose** și **Material 3**, aplicația oferă o interfață vibrantă și interactivă cu animații fluide.
Proiectul urmează arhitectura **MVVM (Model-View-ViewModel)** și utilizează **Koin 4.0** pentru o injecție de dependențe robustă.

## ✨ Caracteristici
- 😈 **Păcălește Troll-ul:** Mecanici de puzzle interactive cu reacții dinamice din partea trollului.
- 🎨 **Material 3 UI:** Design modern și receptiv, bazat pe machete Figma de înaltă fidelitate.
- ⚡ **Animații Avansate:** Mascotă SVG personalizată cu animații de plutire și pulsare.
- 🌓 **Teme Dinamice:** Comutare fluidă între modurile Luminos și Întunecat.
- 🌐 **Acoperire Globală:** Suport complet pentru **25 de limbi** (inclusiv persană, engleză, chineză, rusă, arabă etc.) cu suport automat pentru aspectul RTL.
- 📊 **Monitorizarea Progresului:** Sistem de progresie pe niveluri cu un tablou de bord vizual.
- 🏗️ **Arhitectură Curată:** Construit cu MVVM, StateFlow și Koin 4.0 pentru un standard de dezvoltare premium.
- 💾 **Persistență Robustă:** Utilizarea **Room Database** și **DataStore** pentru stocarea fiabilă a datelor.
