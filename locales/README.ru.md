## 📜 Описание
**TrollMind** — это дерзкая современная игра-головоломка, созданная для того, чтобы бросить вызов вашему остроумию в борьбе с озорным троллем.
Приложение разработано нативно для Android с использованием **Jetpack Compose** и **Material 3**, предлагая яркий и интерактивный интерфейс с плавной анимацией.
Проект следует архитектуре **MVVM (Model-View-ViewModel)** и использует **Koin 4.0** для надежного внедрения зависимостей, что гарантирует чистоту, тестируемость и масштабируемость кодовой базы.

## ✨ Особенности
- 😈 **Перехитри тролля:** Интерактивная механика головоломок с динамической реакцией тролля.
- 🎨 **Интерфейс Material 3:** Современный адаптивный дизайн, основанный на высокоточных макетах Figma.
- ⚡ **Продвинутая анимация:** Кастомный SVG-талисман с анимацией парения и пульсации.
- 🌓 **Динамические темы:** Плавное переключение между светлым и темным режимами.
- 🌐 **Глобальный охват:** Полная поддержка **25 языков** (включая персидский, английский, китайский, русский, арабский и др.) с автоматической поддержкой RTL-раскладки.
- 📊 **Отслеживание прогресса:** Система прогрессии по уровням с визуальной панелью управления.
- 🏗️ **Чистая архитектура:** Создано с использованием MVVM, StateFlow и Koin 4.0 для соответствия премиальным стандартам разработки.
- 💾 **Надежное хранение данных:** Использование **Room Database** и **DataStore** для надежного хранения данных.
