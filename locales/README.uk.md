## 📜 Опис
**TrollMind** (також відомий як **TrapTap**) — це зухвала сучасна гра-головоломка, створена для того, щоб кинути виклик вашій кмітливості у боротьбі з пустотливим тролем.
Розроблено нативно для Android з використанням **Jetpack Compose** та **Material 3**, додаток пропонує яскравий та інтерактивний інтерфейс з плавною анімацією.
Проєкт наслідує архітектуру **MVVM (Model-View-ViewModel)** та використовує **Koin 4.0** для надійного впровадження залежностей.

## ✨ Особливості
- 😈 **Перехитри троля:** Інтерактивна механіка головоломок з динамічною реакцією троля.
- 🎨 **Інтерфейс Material 3:** Сучасний адаптивний дизайн, заснований на високоточних макетах Figma.
- ⚡ **Просунута анімація:** Кастомний SVG-талісман з анімацією паріння та пульсації.
- 🌓 **Динамічні теми:** Плавне перемикання між світлим та темним режимами.
- 🌐 **Глобальне охоплення:** Повна підтримка **25 мов** (включаючи перську, англійську, китайську, російську, арабську тощо) з автоматичною підтримкою RTL-розмітки.
- 📊 **Відстеження прогресу:** Система прогресії за рівнями з візуальною панеллю керування.
- 🏗️ **Чиста архітектура:** Створено з використанням MVVM, StateFlow та Koin 4.0 для відповідності преміальним стандартам розробки.
- 💾 **Надійне збереження даних:** Використання **Room Database** та **DataStore** для надійного зберігання даних.

## 🛠 Зроблено за допомогою

| Категорія | Технологія |
| :--- | :--- |
| 🏛 Архітектура | MVVM (Model-View-ViewModel) |
| 🖼️ UI Framework | [Jetpack Compose](https://developer.android.com/jetpack/compose) (Material 3) |
| 🛠️ Впровадження залежностей | [Koin 4.0](https://insert-koin.io/) |
| 🔄 Управління станом | StateFlow & SharedFlow |

| 🔄 Coroutines | [Kotlinx Coroutines](https://github.com/Kotlin/kotlinx.coroutines) |
| 📦 Серіалізація | [Kotlinx Serialization](https://github.com/Kotlin/kotlinx.serialization) |
| 📦 Локальне сховище | [Room Database](https://developer.android.com/training/data-storage/room) & [DataStore](https://developer.android.com/topic/libraries/architecture/datastore) |
| 🧪 Тестування | [JUnit 4](https://junit.org/junit4/), [MockK](https://mockk.io/), [Turbine](https://github.com/cashapp/turbine) |
