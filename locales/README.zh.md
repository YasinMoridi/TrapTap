## 📜 描述
**特罗尔之心 (TrollMind)** 是一款俏皮、现代的益智游戏，旨在挑战你与调皮特罗尔之间的机智。
该应用专为 Android 构建，使用 **Jetpack Compose** 和 **Material 3**，提供充满活力且互动的用户界面以及流畅的动画。
项目遵循 **MVVM (Model-View-ViewModel)** 架构，并利用 **Koin 4.0** 进行强大的依赖注入，确保代码库整洁、可测试且可扩展。

## ✨ 特色
- 😈 **智取特罗尔：** 具有动态特罗尔反应的互动益智机制。
- 🎨 **Material 3 UI：** 基于高保真 Figma 原型的现代响应式设计。
- ⚡ **高级动画：** 具有浮动和脉冲动画的自定义渲染 SVG 吉祥物。
- 🌓 **动态主题：** 在浅色和深色模式之间无缝切换。
- 🌐 **全球覆盖：** 全面支持 **25 种语言**（包括波斯语、英语、中文、俄语、阿拉伯语等），并支持自动 RTL 布局。
- 📊 **进度追踪：** 带有直观仪表盘的等级进阶系统。
- 🏗️ **整洁架构：** 使用 MVVM、StateFlow 和 Koin 4.0 构建，符合高级开发标准。
- 💾 **稳健持久化：** 利用 **Room Database** 和 **DataStore** 进行可靠的数据存储。

## 🛠 开发技术

| 类别 | 技术 |
| :--- | :--- |
| 🏛 架构 | MVVM (Model-View-ViewModel) |
| 🖼️ UI 框架 | [Jetpack Compose](https://developer.android.com/jetpack/compose) (Material 3) |
| 🛠️ 依赖注入 | [Koin 4.0](https://insert-koin.io/) |
| 🔄 状态管理 | StateFlow & SharedFlow |
| 🧭 导航 | [Compose Navigation](https://developer.android.com/jetpack/compose/navigation) |
| 🔄 协程 | [Kotlinx Coroutines](https://github.com/Kotlin/kotlinx.coroutines) |
| 📦 序列化 | [Kotlinx Serialization](https://github.com/Kotlin/kotlinx.serialization) |
| 📦 本地存储 | [Room Database](https://developer.android.com/training/data-storage/room) & [DataStore](https://developer.android.com/topic/libraries/architecture/datastore) |
| 🧪 测试 | [JUnit 4](https://junit.org/junit4/), [MockK](https://mockk.io/), [Turbine](https://github.com/cashapp/turbine) |
